# Netflix Big Data Analytics using PySpark

## Project Overview

This project demonstrates large-scale data analytics on Netflix content using Apache Spark and PySpark. The objective was to process and analyze a large entertainment dataset efficiently while extracting meaningful business insights related to content distribution, genres, countries, ratings, and release trends.

## Business Problem

Streaming platforms generate massive volumes of content metadata. Understanding content distribution and audience preferences is essential for strategic content acquisition and production decisions.

This project leverages distributed computing techniques to analyze Netflix data at scale and uncover actionable insights.

## Technologies Used

* PySpark
* Spark SQL
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Key Features

* Large-scale data processing using PySpark
* Data cleaning and preprocessing
* Missing value treatment
* Feature engineering
* Spark SQL analysis
* Distributed aggregations
* Window function implementation
* Trend analysis and visualization

## Analysis Performed

* Content type distribution
* Genre popularity analysis
* Country-wise content production
* Release year trends
* Rating distribution analysis
* Top contributing countries
* Content growth patterns over time

## Key Insights

* Identified dominant content categories on Netflix.
* Analyzed global content production trends.
* Evaluated rating distributions and audience segmentation.
* Uncovered regional differences in content creation.

## Business Impact

The analysis demonstrates how big data technologies can support strategic decision-making for content recommendation, licensing, and production planning.
