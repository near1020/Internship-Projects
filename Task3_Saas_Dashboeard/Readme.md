# SaaS Sales Performance Dashboard | Power BI

## Project Overview

This project presents an executive-level SaaS Sales Performance Dashboard built in Power BI. The dashboard enables stakeholders to monitor business performance, customer behavior, product performance, and revenue trends through interactive visualizations.

## Business Problem

SaaS companies rely on data-driven decision-making to optimize customer acquisition, retention, and revenue growth. Executives require consolidated reporting for quick strategic assessments.

* Power Query
